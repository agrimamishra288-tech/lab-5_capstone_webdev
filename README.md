## Project Overview

This project is a responsive website made using HTML5 and CSS3.

The website is designed for a real-world theme.

It works on desktop, tablet, and mobile screens.

--------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------

## Project Theme

Theme Chosen: Travel Agency Website

--------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------

## Website Sections

The website includes the following sections:

- Header with logo and navigation menu

- Hero section with banner and button

- Tour package cards shown in a grid

- Reviews / testimonials section

- Contact or inquiry form

- Footer with basic information

- Technologies Used

- HTML5 (semantic tags)

- CSS3

- Flexbox and Grid

- Media Queries

- Responsive Design


The website is responsive and works on:


1). Desktop (1024px and above)

2). Tablet (around 768px)

3). Mobile (600px and below)

4). Layouts change automatically for smaller screens.

--------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------

## What I Learned

- Using semantic HTML tags

- Creating layouts with Flexbox and Grid

- Making websites responsive

- Using media queries

- Designing a real-world website
